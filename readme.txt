== Description ==

Wiziva is a packing and installation platform for WordPress.<br />
Within the Wiziva Dashboard you can manage a portfolio of your favorite plugins and themes and easily install them in bulk.<br />
Optionally you can put all that in the cloud with a Wiziva.com (free) account and access your portfolio from every WordPress installation you may have.<br />

== Installation ==

Installation Option 1:
-----------------------
1. Download the file (wiziva.zip) to your computer
2. Login to your Wordpress Admin Area
3. Navigate to Plugins - Install Plugins - Add New - Upload
4. Upload the file, you saved on step 1. and when asked activate the plugin

Installation Option 2:
-----------------------
1. Login to your Wordpress Admin Area
2. Navigate to Plugins - Install Plugins
3. Search for "Wiziva", in the search result find the exact title and click "Install"
4. When the installation finish click the "Activate Plugin" link in the installation report page or in the plugins list

== Frequently Asked Questions ==

= Are there any limits on the number of plugins and themes I can manage? =

No, there is absolutely no limit.<br />
You can import and manage as many plugins and themes as there are currently in the WordPress repository.


= May I upload my own plugins to the Wiziva Dashboard? =

In the current version this option is not available, but you can upload plugins and themes to your Wiziva.com account
and then import them to the Dashboard.
